# Tier List Maker

A lightweight, single-page tier list maker that runs entirely in the browser — no installation, no account, no data sent anywhere.

## Features

- Upload images and drag them into ranked tiers (S, A, B, C, D by default)
- **Click an image** to zoom it fullscreen — click again to dismiss
- Add, edit, reorder, and delete tiers with custom names and colors
- Sort or shuffle the unranked item pool
- Export your tier list as a PNG image
- Works on desktop and mobile

## Usage

Visit the live app: **https://drquadfather.github.io/tier-list-maker/**

Or open `index.html` directly in any browser — no server needed.
